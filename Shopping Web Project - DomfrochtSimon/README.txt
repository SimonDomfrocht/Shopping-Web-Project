Academic PHP Shopping Web Project – XAMPP (MariaDB)

This repository contains an academic project developed as part of a university course.
The application was built using PHP and MariaDB, running on a local server environment
such as XAMPP.

Requirements

-PHP 8.x (or compatible)

-MariaDB (included in XAMPP)

-Local server environment (XAMPP, WAMP, MAMP, etc.)

-Web browser

Installation and Setup

1- Clone or download this repository.

2- Copy the project folder into the XAMPP htdocs directory: "xampp/htdocs/"

3- Open the XAMPP Control Panel and start Apache and MariaDB.

4- Open phpMyAdmin in your browser and import the database:

5- Create a database if it does not exist.

6- Import the SQL file named labo2.sql included in this repository.

7- Make sure the database connection settings in the PHP files match your local MariaDB configuration.

8- Open the project in your browser: "http://localhost/Shopping Web Project - DomfrochtSimon"

Database

Database engine: MariaDB

- The database structure is provided in the file labo2.sql

- This file must be imported for the project to work correctly.

Project Description

This project is an academic exercise focused on:

- PHP programming

- Handling forms and user input

- Basic interaction with a MariaDB database

- Applying programming logic and control structures
